# Resources

This folder holds reusable reference material that is worth keeping in Betty Brain.

## What belongs here

- research notes
- comparative studies
- domain references
- reusable templates
- cross-project technical references
- important non-project-specific reports

## Report storage rule

For important reports, prefer a folder-per-report structure:

- `7 - Resources/reports/<report-name>/report.md`
- `7 - Resources/reports/<report-name>/index.html`

This is cleaner than scattering standalone report files across the root of `7 - Resources/`.

## Example

- `7 - Resources/reports/agent-memory-youtube-study-2026-04-23/`

Use `7 - Resources/` for reusable material.
Use `5 - Projects/<Project>/reports/` when the report is tightly tied to one project.
