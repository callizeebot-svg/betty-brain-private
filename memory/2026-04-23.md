## Research and memory architecture direction

- Ran a YouTube research sweep on agent memory and compared the recurring patterns with Betty Brain.
- Durable takeaway: the strongest external pattern is layered memory (short-term/session state + long-term persistent memory + compaction/retrieval triggers), which matches Betty's direction but highlights gaps in live semantic retrieval, explicit runtime state handling, and fully operational memory-maintenance loops.
