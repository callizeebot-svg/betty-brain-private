# Daily Journal

## Purpose

This folder is for day-level working memory.

Use it for the rawer, more reflective, more human view of what a day contained:

- thoughts worth revisiting
- rough observations
- emotional texture
- loose ideas
- day narratives
- things that may later matter but are not yet shaped

## Daily Journal vs Live Logs

These are not the same thing.

### Daily Journal

Think:

- what the day felt like
- what was noticed
- what might matter later
- what still feels unresolved

Examples:

- "I keep avoiding this project and should understand why."
- "This meeting felt off even though nothing obviously broke."
- "I may have found a better product angle."

### Live Logs

Think:

- what happened
- what changed
- what was decided
- what Betty or Cally actually did

Examples:

- a workflow was added
- a system setting changed
- a decision became durable
- a meaningful assistant action was completed

## Relationship

Good flow:

`Daily Journal -> review/process -> Live Logs / Projects / MEMORY / Areas / People`

The journal is allowed to be messier.
It is not required to be the final canonical home.

## What belongs here

- day notes
- reflective fragments
- personal observations
- unresolved questions
- rough planning thoughts
- "this might matter" material

## What does not belong here as the final home

- durable project state
- stable operational rules
- core identity updates
- important assistant actions that should be logged operationally

Those should be promoted when they become durable.

## Writing rules

- date entries clearly
- prefer honesty over polish
- do not force structure too early
- avoid storing secrets directly
- when something becomes durable, promote it instead of duplicating it forever

## Processing rule

If a journal entry changes how Betty should think or operate, process it onward:

- operational change -> `9 - Operations/`
- dated event / decision / action -> `2 - Live Logs/`
- durable personal memory -> `MEMORY.md` or `memory/YYYY-MM-DD.md`
- project truth -> `5 - Projects/`

The journal is not wasted space.
It is the raw day-stream that can later become stronger memory.
