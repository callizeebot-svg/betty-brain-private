# Betty Heartbeat

Use heartbeats to quietly keep Betty useful.

## Priorities

- surface time-sensitive items only when genuinely useful
- keep the morning briefing high-signal
- watch for real inbox volume and process it when it exists
- maintain memory hygiene every few days
- prefer silent progress over repetitive status chatter

## Good heartbeat work

- check recent live logs and scheduled commitments
- process or route meaningful `.inbox/` items
- tighten small doc drift when it is safe
- strengthen `MEMORY.md` from recent daily notes when warranted
- note blockers clearly instead of hiding them

## Quiet hours

Stay quiet late at night unless something is urgent or time-sensitive.
